# Method Notes

## V1

V1 使用人工控制点完成初始化，关注建筑周边地面区域的连续可观察性。

核心流程：

1. 在每路图像中手工标注地面控制点。
2. 将控制点映射到统一全局画布。
3. 使用 `cv2.findHomography` 估计每路图像到画布的平面变换。
4. 将建筑区域单独标注为 mask，并在融合阶段排除出地面权重。
5. 使用简单融合方法得到全局监控图。
